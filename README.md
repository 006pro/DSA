# DSA Practice

This repository is for practicing Data Structures and Algorithms problems and keeping solutions organized in one place.

## Goal

The main purpose of this repo is to:
- solve DSA problems regularly
- understand the logic behind each solution
- improve problem-solving speed and accuracy
- build a strong foundation in core coding concepts

## Topics to Cover

- Arrays
- Strings
- Linked List
- Stack and Queue
- Hashing
- Trees
- Graphs
- Recursion
- Dynamic Programming
- Sorting and Searching
- Heaps and Priority Queues

## Suggested Structure

Each topic can have its own folder with problem solutions, for example:

- arrays/
- strings/
- linked-list/
- trees/
- graphs/
- dynamic-programming/

## How to Use This Repository

1. Choose a topic.
2. Solve a problem.
3. Add your solution with a clear explanation.
4. Mention the time and space complexity.
5. Keep updating the progress tracker.

## Solution Format

For every problem, try to include:
- Problem name
- Approach
- Algorithm steps
- Time complexity
- Space complexity
- Code
